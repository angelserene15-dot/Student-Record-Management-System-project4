**Student Record Management System**

**Project Overview:**

The Student Record Management System is a console-based application developed in C that helps manage student information efficiently. The system allows users to add, view, search, update, and delete student records through a menu-driven interface.

Student data such as Roll Number, Name, Branch, and Marks are stored permanently using file handling techniques, ensuring that records remain available even after the program is closed. This project demonstrates the practical implementation of data management concepts and serves as an excellent example of real-world applications developed using the C programming language.
